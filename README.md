#  Hotel Management System

A final-year BCA project developed using **Java (Swing & AWT)** for the frontend and **MySQL** as the backend database.

# Project Overview  
This desktop application streamlines hotel operations like room booking, check-in/check-out, and employee/customer management. Built with Java Swing for GUI and MySQL for database, itâ€™s designed for academic and real-world learning purposes.

# Features  
- Add, view, and delete rooms  
- Customer check-in and check-out  
- Room status updates  
- Employee and customer management  
- Checkout summary with current status

# Technologies Used  
- Java (Core Java, Swing & AWT)  
- MySQL  
- JDBC (Java Database Connectivity)  
- NetBeans IDE
